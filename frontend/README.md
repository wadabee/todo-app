# Frontend
## Environments

### Requirements

The following must be installed.
* Node.js  
  Recommend: Install via version manager tool for node.js (e.g. nvm, nodebrew, etc...)


### Get started

- install node modules

```Bash
npm ci
```

## Application Launch

```bash
# communicate with backend
npm start

# comunicate with mock(using MSW)
npm run mock
```
